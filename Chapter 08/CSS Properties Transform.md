## 💡 변환
---
- ❗ transform
  - 요소의 **변환 효과**
  - ```CSS
    transform: 변환함수1 변환함수2 변환함수3 ...;
    transform: 원근법 이동 크기 회전 기울임;
    ```
  - 2D 변환 함수
  - **단위 px**
    - translate(x, y)
      - 이동(x축, y축)
    - translateX(x)
      - 이동(x축)
    - translateY(y)
      - 이동(y축)
  - **없음(배수)**
    - scale(x, y)
      - 크기(x축, y축)
  - **단위 deg**
    - rotate(degree)
      - 회전(각도)
    - skewX(x)
      - 기울임(x축)
    - skewY(y)
      - 기울임(y축)
  - matrix
    - matrix(n, n, n, n, n, n)
      - 2차원 변환 효과
  - 3D 변환 함수
