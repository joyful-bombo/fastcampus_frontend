## 💡 문서의 전체 범위
```
<html> 

</html>
```
- HTML 문서가 어디에서 시작하고, 어디에서 끝나는지 알려주는 역할

## 💡 문서의 **정보**를 나타내는 범위
```
<head> 

</head>
```
- 웹 브라우저가 해석해야 할 웹 페이지의 제목, 설명, 사용할 파일 위치, 스타일(CSS)같은 웹 페이지의 **보이지 않는 정보**를 작성하는 범위

## 💡 문서의 **구조**를 나타내는 범위
```
<body> 

</body>
```
- 사용자 화면을 통해 보여지는 로고, 헤더, 푸터, 내비게이션, 메뉴, 버튼, 이미지 같은 웹페이지에 **보여지는 구조**를 작성하는 범위
