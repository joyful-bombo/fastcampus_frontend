## 💡 HTML 핵심 Element 정리

## 💡 Block Element
  - div(Division)
    - 특별한 의미가 없는 구분을 위한 요소.
  - h1(Heading) 1 ~ 6
    - 제목을 의미하는 요소
    - likely. 목차
    - 숫자가 **작을 수록 더 중요한 제목**을 정의
  - p(Paragraph)
    - 문장을 의미하는 요소
  - ul(Unordered List)
    - 순서가 필요없는 목록의 집합을 의미
  - ol(ordered List) <-- 빈도가 적음
    - 순서가 정해져있는 목록의 집합을 의미 
  - li(List item)
    - 목록 내 각 항목
  
## 💡 Inline Element
  - img(Image)
    - 이미지를 삽입하는 요소
  - a(Anchor)
    - 다른/같은 페이지로 이동하는 하이퍼링크를 지정하는 요소
    - target(Attribute) 
      - _blank : 새탭으로 열기
  - span
    - 특별한 의미가 없는 구분을 위한 요소
  - br(Break)
    - 줄바꿈 요소
  - label
    - 라벨 가능 요소(input)의 제목
    
## 💡 Inline-Block Elemnet
  - **가로, 세로 사이즈 및 여백 설정 가능, 하지만 수평으로 요소 채워짐.
  - input
    - 사용자가 데이터를 입력하는 요소
    - type(Attribute)
      - text, checkbox 등등
    - value(Attribute)
      - 미리 입력된 값
    - placeholder(Attribute)
      - 사용자가 입력할 값의 힌트
    - disabled(Attribute)
      - 입력 요소 비활성화
    - checkbox(Attribute)
      - checked(Attribute)
        - 입력 요소 체크 됌
    - radio(Attribute)
      - 사용자에게 체크 여부를 그룹에서 1개만 입력 받음!(택1)
      - name(Attribute)
        - 그룹

## 💡 Table Element <- 블록에 포함
  - 표 요소, 행(Row)과 열(Column)의 집합.
  - tr(Table Row)
    - 행(Row)을 지정하는 요소.
  - td(Table Data)
    - 열(Column)을 지정하는 요소
