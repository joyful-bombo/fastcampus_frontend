## 💡 CSS 기본 문법
```css
<div>HEROPY</div>

div {
  font-size: 50px;
  color: blue;
  text-decoration: underline;
}
```
- 선택자{ 속성 : 값; 속성 : 값; }
  - 스타일(CSS)을 적용할 **대상** (Selector)
  - 속성
    - 스타일(CSS)의 **종류** (Property)
  - 값
    - 스타일(CSS)의 **값** (Value)
  - **속성은 값이다.**
  - **{** : 스타일 범위의 시작, **}** : 스타일 범위의 끝
  
- margin
  - 요소 외부 여백
- 주석
  - /* 주석입니다. */
