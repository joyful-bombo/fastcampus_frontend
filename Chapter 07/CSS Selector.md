# 💡 CSS 선택자

## 💡 기본
  - 전체 선택자(Universal Selector)
    - **모든** 요소를 선택
    - ```
      *{
        color: red;
      }
      ```
  - 태그 선택자(Type Selector)
    - **태그 이름** 요소 선택
    - ```
      li{
        color: red;
      }
      ```
  - 클래스 선택자(Class Selector)
    - HTML **class 속성의 값**인 요소를 선택
    - ```
      .orange{
        color: red;
      }
      ```
  - 아이디 선택자(ID Selector)
  - ```
    #orange{
      color: red;
    }
    ```
