## 💡 Front-end development
- HTML, CSS, JS를 사용해 데이터를 GUI로 변환하고, 사용자와 상호 작용할 수 있도록 하는 것

### Business Logic
- 웹 사이트가 동작하는데 필요한 핵심 데이터 처리를 수행하는 알고리즘

### 💡 Full-stack development
- Front-end 부터 Back-end 까지 모든 소프트웨어 스택을 개발하는 것
